# Translator-Storage
Translate website and store vocabulary.

## About the Project
**Translator-Storage** is an intuitive web application designed to help users instantly translate content from websites and effectively store new vocabulary. 

Whether you are reading articles, studying, or browsing foreign websites, this tool provides a seamless translation experience while allowing you to save unfamiliar words or phrases to your personal dictionary for future review.
